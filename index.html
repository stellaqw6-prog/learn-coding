🖨️ Input & Output

📤 Output dengan print()
Fungsi print() digunakan untuk menampilkan teks atau nilai ke layar. Kamu bisa cetak teks, angka, variabel, atau kombinasinya.
output.py 
nama = "Budi"
umur = 18

# Cara 1: pakai koma

print(f"Halo, {nama}!")

🔁 Perulangan (Loop)
Loop membuat kode dijalankan berulang kali secara otomatis. Bayangkan kamu disuruh tulis angka 1 sampai 100 — dengan loop, 1 baris kode bisa melakukannya!
🔄 For Loop
for loop digunakan ketika kamu tahu berapa kali perulangan dilakukan.

range(n) menghasilkan angka dari 0 sampai n-1.
range(1, n) menghasilkan angka dari 1 sampai n-1.
range(1, n, 2) menghasilkan angka dari 1 sampai n-1, loncat 2.
forloop.py 
# Cetak angka 1 sampai 5
for i in range(1, 6):
    print(f"Angka: {i}")

# Loop melalui list
buah = ["apel", "mangga", "jeruk"]
for b in buah:
    print(f"Saya suka {b}")

# Tabel perkalian 3
for x in range(1, 11):
    print(f"3 x {x} = {3 * x}")
🔂 While Loop
while loop digunakan ketika perulangan terus berjalan selama kondisi masih True. Hati-hati: kalau kondisinya tidak pernah False, program akan jalan terus (infinite loop)!
whileloop.py 
hitung = 1

while hitung <= 5:
    print(f"Hitungan ke-{hitung}")
    hitung += 1   # PENTING! Tanpa ini → infinite loop

print("Selesai!")
hitung += 1 artinya sama dengan hitung = hitung + 1 — tambahkan 1 ke nilai sekarang.

⚙️ Fungsi (Function)
Fungsi adalah blok kode yang diberi nama dan bisa dipanggil kapan saja. Prinsipnya: tulis sekali, gunakan berkali-kali!
📖 Struktur Fungsi
Fungsi dibuat dengan kata kunci def, diikuti nama fungsi, lalu tanda kurung () dan titik dua :.

Fungsi bisa menerima parameter (input) dan mengembalikan return value (output/hasil).
fungsi.py 
# Fungsi sederhana tanpa parameter
def sapa():
    print("Halo! Selamat datang!")

sapa()   # Memanggil fungsi

# Fungsi dengan parameter
def sapa_nama(nama):
    print(f"Halo, {nama}! Selamat belajar!")

sapa_nama("Asus")
sapa_nama("Budi")

# Fungsi dengan return (mengembalikan nilai)
def tambah(a, b):
    hasil = a + b
    return hasil

angka = tambah(5, 3)
print(f"5 + 3 = {angka}")

📋 List & Dictionary
Sampai sekarang kita menyimpan 1 data per variabel. Bagaimana jika ingin simpan 100 data? Gunakan List dan Dictionary!
📝 List (Daftar)
List adalah kumpulan data dalam urutan tertentu, ditulis dalam tanda kurung siku [].

Setiap item punya index (posisi) dimulai dari 0. Jadi item pertama index-nya 0, kedua index 1, dst.
list.py 
buah = ["apel", "mangga", "jeruk", "pisang"]

print(buah[0])       # apel (index pertama = 0)
print(buah[2])       # jeruk
print(buah[-1])      # pisang (index -1 = terakhir)

# Tambah item
buah.append("anggur")
print(buah)

# Panjang list
print(f"Jumlah buah: {len(buah)}")

# Loop semua item
for b in buah:
    print(f"- {b}")
🗂️ Dictionary (Kamus)
Dictionary menyimpan data dalam pasangan key: value, ditulis dalam kurung kurawal {}.

Analoginya seperti KTP: nama → "Budi", umur → 20, kota → "Jakarta". Kamu ambil data berdasarkan nama kunci, bukan nomor urut.
dictionary.py 
mahasiswa = {
    "nama": "Asus",
    "umur": 20,
    "jurusan": "Teknik Informatika",
    "ipk": 3.75
}

# Ambil data
print(mahasiswa["nama"])
print(mahasiswa["ipk"])

# Tambah data baru
mahasiswa["kota"] = "Bandung"

# Loop semua key-value
for key, value in mahasiswa.items():
    print(f"{key}: {value}")

kalkulator.py 
# ===== KALKULATOR PYTHON =====
# Gabungan semua materi yang sudah dipelajari

# Fungsi-fungsi operasi matematika
def tambah(a, b):
    return a + b

def kurang(a, b):
    return a - b

def kali(a, b):
    return a * b

def bagi(a, b):
    if b == 0:
        return "Error: Tidak bisa bagi dengan 0!"
    return a / b

# Program utama
print("================================")
print("     KALKULATOR PYTHON v1.0     ")
print("================================")

while True:
    # Input angka
    angka1 = float(input("\nAngka pertama: "))
    angka2 = float(input("Angka kedua: "))

    # Tampilkan menu
    print("\nPilih operasi:")
    print("1. Penjumlahan (+)")
    print("2. Pengurangan (-)")
    print("3. Perkalian (*)")
    print("4. Pembagian (/)")

    pilihan = input("Pilihan (1-4): ")

    # Proses pilihan
    if pilihan == "1":
        hasil = tambah(angka1, angka2)
        op = "+"
    elif pilihan == "2":
        hasil = kurang(angka1, angka2)
        op = "-"
    elif pilihan == "3":
        hasil = kali(angka1, angka2)
        op = "*"
    elif pilihan == "4":
        hasil = bagi(angka1, angka2)
        op = "/"
    else:
        print("Pilihan tidak valid!")
        continue

    # Tampilkan hasil
    print(f"\n{angka1} {op} {angka2} = {hasil}")

    # Tanya lanjut atau tidak
    lagi = input("\nHitung lagi? (y/n): ")
    if lagi != "y":
        print("Terima kasih! Sampai jumpa!")
        break

📦 Variabel & Tipe Data JavaScript
Di JavaScript ada 3 cara buat variabel: let, const, dan var. Tenang, kita bahas satu per satu dengan santai!
🗃️ let, const, dan var — Apa Bedanya?
Bayangkan kamu punya 3 jenis kotak:

📦 let = Kotak biasa yang bisa diganti isinya. Paling sering dipakai!
🔒 const = Kotak yang dikunci. Sekali diisi, tidak bisa diganti. Untuk data yang tetap.
📫 var = Kotak jadul dari zaman dulu. Masih jalan, tapi tidak disarankan lagi.

Aturan praktis: Gunakan const kalau datanya tidak berubah, let kalau datanya bisa berubah.
variabel.js 
// let — bisa diubah nilainya
let nama = "Asus"
nama = "Budi"  // boleh diubah

// const — TIDAK bisa diubah
const PI = 3.14
// PI = 3 → ERROR! const tidak bisa diubah

// Tipe data otomatis terdeteksi
let teks = "Ini String"      // String
let umur = 20              // Number (int & float gabung!)
let tinggi = 1.75          // Number juga
let lulus = true           // Boolean (huruf kecil!)
let kosong = null          // Kosong / tidak ada

console.log(nama, umur, tinggi, lulus)
🆚 Beda dengan Python:
Python: nama = "Budi"
JavaScript: let nama = "Budi"

Di JS harus ada kata kunci let atau const di depan. Di Python tidak perlu.
🧮 Operasi Matematika JS
Sama persis dengan Python! + - * / % — bedanya pangkat pakai ** juga, dan ada satu trik keren: += untuk tambah sekaligus simpan.
matematika.js 
let a = 10
let b = 3

console.log(a + b)   // 13
console.log(a - b)   // 7
console.log(a * b)   // 30
console.log(a / b)   // 3.333...
console.log(a % b)   // 1 (sisa bagi)
console.log(a ** b)  // 1000 (pangkat)

// Shortcut yang keren

console.log("Skor:", skor)

🖨️ Output & Template Literal

📤 console.log() — Si Print JavaScript

🧩 Template Literal — Cara Modern & Keren!
Template Literal di JavaScript = f-string di Python. Cara memasukkan variabel langsung ke dalam teks.

🐍 Python: f"Halo {nama}" → pakai huruf f + tanda kutip biasa
⚡ JavaScript: `Halo ${nama}` → pakai backtick (tanda ` di atas tombol Tab) + tanda dollar

Perhatikan: Python pakai {}, JavaScript pakai ${}
template.js 
let nama = "Asus"
let umur = 20
let kota = "Jakarta"

// Template Literal — pakai backtick `
console.log(`Halo, nama saya ${nama}`)
console.log(`Saya berumur ${umur} tahun`)
console.log(`Tinggal di ${kota}`)

// Bisa langsung hitung di dalamnya!
console.log(`Tahun depan saya berumur ${umur + 1} tahun`)
console.log(`10 x 5 = ${10 * 5}`)
Backtick (`) letaknya di keyboard ada di pojok kiri atas, di bawah tombol Esc, atau samping angka 1. Biasanya perlu Shift untuk menekannya di keyboard Indonesia.

🔀 Kondisi if / else JavaScript
Sama persis seperti Python — program bisa "berpikir" dan memilih jalan yang berbeda. Yang beda hanya penulisannya pakai kurung kurawal { }
📖 Cara Tulis if/else di JS
🐍 Python pakai indentasi (spasi) untuk tahu batas blok kode
⚡ JavaScript pakai kurung kurawal { } untuk tahu batas blok kode

Selain itu, kondisi di JS ditulis dalam tanda kurung ( ) setelah kata if.
kondisi.js 
let nilai = 75

// if/else biasa
if (nilai >= 75) {
    console.log("Selamat, kamu LULUS!")
} else {
    console.log("Maaf, kamu tidak lulus.")
}

// else if (bukan elif seperti Python!)
if (nilai >= 90) {
    console.log("Grade: A")
} else if (nilai >= 80) {
    console.log("Grade: B")
} else if (nilai >= 70) {
    console.log("Grade: C")
} else {
    console.log("Grade: D")
}
🆚 Perbedaan Python vs JavaScript:
Python: elif → JavaScript: else if (2 kata!)
Python: blok ditandai spasi → JavaScript: blok ditandai { }
🔄 Switch Case — Alternatif if/else
switch adalah cara elegan jika kamu punya banyak pilihan yang dibandingkan dengan satu variabel. Seperti remote TV — satu tombol, banyak pilihan!
switch.js 
let hari = "Senin"

switch (hari) {
    case "Senin":
        console.log("Hari pertama, semangat!")
        break
    case "Jumat":
        console.log("Besok weekend! 🎉")
        break
    case "Sabtu":
    case "Minggu":

    default:
        console.log("Hari biasa, tetap produktif!")
}
Selalu tulis break di akhir setiap case, biar program berhenti dan tidak lanjut ke case berikutnya.

🔁 Loop & Perulangan JavaScript
JavaScript punya beberapa cara melakukan perulangan. Ada yang mirip Python, ada yang khas JS. Semua akan kita pelajari!
🔄 For Loop — Klasik
For loop di JS sedikit beda dengan Python. Di JS kamu mendefinisikan:
1️⃣ Titik mulai (let i = 0)
2️⃣ Kondisi berhenti (i < 5)
3️⃣ Cara naik/turun (i++ = tambah 1 setiap putaran)

Semuanya ditulis dalam satu baris, dipisah titik koma.
forloop.js 
// Cetak angka 1 sampai 5
for (let i = 1; i <= 5; i++) {
    console.log(`Angka: ${i}`)
}

// Tabel perkalian 3
for (let x = 1; x <= 10; x++) {
    console.log(`3 x ${x} = ${3 * x}`)
}
🆚 Python: for i in range(1, 6):
⚡ JavaScript: for (let i = 1; i <= 5; i++)

Lebih panjang, tapi lebih fleksibel! Kamu bisa atur sendiri mau mulai dari mana, berhenti kapan, dan melompat berapa.
🔂 While Loop
While loop di JavaScript sama persis polanya dengan Python, cuma sintaksnya sedikit beda: kondisi ditulis dalam tanda kurung dan blok kode dibungkus { }.
while.js 
let hitung = 1

while (hitung <= 5) {
    console.log(`Hitungan ke-${hitung}`)
    hitung++   // WAJIB! Tanpa ini → infinite loop
}

console.log("Selesai!")

Fungsi di JavaScript mirip Python — tulis sekali, pakai berkali-kali. Tapi JS punya cara penulisan tambahan yang lebih modern: Arrow Function!
📖 Cara Buat Function
Di Python pakai def, di JavaScript pakai kata kunci function. Blok kode dibungkus kurung kurawal { }.
function.js 
// Fungsi sederhana
function sapa() {
    console.log("Halo! Selamat datang!")
}
sapa()  // Panggil fungsi

// Fungsi dengan parameter
function sapaNama(nama) {
    console.log(`Halo, ${nama}! Selamat belajar!`)

sapaNama("Asus")
sapaNama("Budi")

// Fungsi dengan return (mengembalikan nilai)
function tambah(a, b) {
    return a + b

🏹 Arrow Function — Cara Modern
Arrow Function adalah cara penulisan fungsi yang lebih singkat dan modern. Banyak dipakai di JavaScript zaman sekarang. Konsepnya sama, cuma tulisannya lebih pendek!

Dari: function tambah(a, b) { return a + b }
Jadi: const tambah = (a, b) => a + b

Tanda => dibaca "arrow" (panah), makanya disebut Arrow Function.
arrow.js 
// Cara lama
function kuadrat(x) {
    return x * x
}

// Arrow function — cara modern
const kuadrat2 = (x) => x * x

// Keduanya hasilnya sama!
console.log(kuadrat(5))    // 25
console.log(kuadrat2(5))   // 25

// Arrow function dengan beberapa baris
const greet = (nama, umur) => {
    console.log(`Nama: ${nama}`)
    console.log(`Umur: ${umur}`)
}
greet("Asus", 20)
🎮 Latihan Function
▶ LATIHAN
Perhatikan contoh di atas: ada yang pakai function biasa, ada yang pakai arrow function =>. Keduanya benar! Di dunia kerja kamu akan sering lihat keduanya.
🧠 Kuis: Versi arrow function dari function kali(a,b) { return a*b } yang benar adalah?
Benar! Arrow function selalu disimpan ke variabel dengan const atau let, lalu pakai tanda => (sama dengan lebih dari) untuk memisahkan parameter dan body fungsi. ✅
📋 Array & Object JavaScript
Array = List-nya Python. Object = Dictionary-nya Python. Konsep sama, sintaks beda. Ini adalah dasar dari hampir semua aplikasi nyata!
📝 Array — Kumpulan Data Berurutan
Array di JavaScript persis seperti List di Python — ditulis dengan tanda kurung siku [], bisa menyimpan banyak data, dan diakses dengan index mulai dari 0.

Bedanya: Array JS punya banyak method (fungsi bawaan) yang sangat berguna!
array.js 
const buah = ["apel", "mangga", "jeruk", "pisang"]

console.log(buah[0])         // apel
console.log(buah[2])         // jeruk
console.log(buah.length)     // 4 (panjang array)

// Tambah item di akhir
buah.push("anggur")
console.log(buah)

// Loop dengan forEach — cara modern!
buah.forEach((b) => {
    console.log(`- ${b}`)
})

// Loop biasa juga bisa
for (let i = 0; i < buah.length; i++) {
    console.log(buah[i])
}
forEach adalah cara paling modern untuk loop array di JS. Kamu kasih fungsi, dia jalankan untuk setiap item. Cara ini banyak dipakai di React, Vue, dan framework modern lainnya.
🗂️ Object — Data Berstruktur dengan Nama
Object di JavaScript = Dictionary di Python. Menyimpan data dalam pasangan key: value, ditulis dalam kurung kurawal {}.

Kalau di Python ambil data pakai dict["key"], di JS bisa pakai obj["key"] ATAU obj.key (pakai titik — lebih umum!).
object.js 
const mahasiswa = {
    nama: "Asus",
    umur: 20,
    jurusan: "Teknik Informatika",
    ipk: 3.75
}

// Ambil data — pakai titik (dot notation)
console.log(mahasiswa.nama)
console.log(mahasiswa.ipk)

// Atau pakai bracket
console.log(mahasiswa["jurusan"])

// Tambah properti baru
mahasiswa.kota = "Bandung"

// Loop semua properti
for (let key in mahasiswa) {
    console.log(`${key}: ${mahasiswa[key]}`)
}

🌐 JavaScript di Web — Manipulasi DOM
Inilah kekuatan terbesar JavaScript yang tidak dimiliki Python: bisa langsung mengubah tampilan halaman web tanpa reload! Inilah yang membuat website jadi interaktif.
🤔 Apa itu DOM?
DOM (Document Object Model) adalah cara JavaScript "melihat" dan "memegang" sebuah halaman HTML.

Bayangkan halaman web adalah sebuah pohon:
🌳 document = seluruh pohon (halaman)
🌿 element = setiap ranting (tag HTML: div, button, h1, dll)
🍃 text = daun (isi teks di dalam tag)

JavaScript bisa ambil ranting mana saja, ganti teksnya, ubah warnanya, hapus, atau tambah ranting baru — semua secara langsung!
🎯 Ambil & Ubah Element HTML
dom.html 
<!-- HTML: buat elemen dengan id -->
<h1 id="judul">Teks Awal</h1>
<p id="paragraf">Paragraf awal</p>
<button onclick="ubahTeks()">Klik saya!</button>

<script>
function ubahTeks() {
    // Ambil elemen berdasarkan id
    const judul = document.getElementById("judul")
    const para = document.getElementById("paragraf")

    // Ubah teksnya
    judul.textContent = "Teks Sudah Berubah! 🎉"
    para.textContent = "Paragraf ini diubah oleh JavaScript!"

    // Ubah warnanya
    judul.style.color = "#00e5a0"
}
</script>
Perintah DOM yang paling sering dipakai:

📌 document.getElementById("id") — ambil element berdasarkan id
📌 element.textContent = "teks" — ganti teks di dalam element
📌 element.style.color = "red" — ganti warna teks
📌 element.style.display = "none" — sembunyikan element
📌 element.classList.add("nama-class") — tambahkan CSS class

📄 Kode Lengkap To-Do List
todo.html 
<!DOCTYPE html>
<html>
<head>
  <title>To-Do List</title>
  <style>
    body { font-family: Arial; max-width: 400px; margin: 40px auto; padding: 20px; }
    h1 { color: #f7df1e; text-align: center; }
    .input-area { display: flex; gap: 8px; margin-bottom: 20px; }
    input { flex:1; padding: 10px; border: 2px solid #ddd; border-radius: 8px; font-size: 1rem; }
    button { padding: 10px 16px; background: #f7df1e; border: none; border-radius: 8px; cursor: pointer; font-weight: bold; }
    .task { display: flex; align-items: center; gap: 10px; padding: 12px; margin: 6px 0;
            background: #f9f9f9; border-radius: 8px; border: 1px solid #eee; }
    .task.done span { text-decoration: line-through; color: #999; }
    .task button { padding: 4px 10px; background: #ff6b6b; color: white; font-size: 0.8rem; }
    #info { text-align: center; color: #666; margin-top: 16px; }
  </style>
</head>
<body>
  <h1>📝 To-Do List</h1>

  <div class="input-area">
    <input id="inputTugas" type="text" placeholder="Tulis tugas...">
    <button onclick="tambahTugas()">Tambah</button>
  </div>

  <div id="daftarTugas"></div>
  <p id="info"></p>

  <script>
    let tugas = []   // Array untuk menyimpan semua tugas

    function tambahTugas() {
      const input = document.getElementById("inputTugas")
      const teks = input.value.trim()

      if (teks === "") return   // Jika kosong, tidak tambah

      tugas.push({ teks: teks, selesai: false })
      input.value = ""   // Kosongkan input
      render()
    }

    function toggleSelesai(index) {
      tugas[index].selesai = !tugas[index].selesai
      render()
    }

    function hapusTugas(index) {
      tugas.splice(index, 1)   // Hapus 1 item dari index
      render()
    }

    function render() {
      const daftar = document.getElementById("daftarTugas")
      daftar.innerHTML = ""   // Kosongkan dulu

      tugas.forEach((t, i) => {
        daftar.innerHTML += `
          <div class="task ${t.selesai ? 'done' : ''}">
            <input type="checkbox" ${t.selesai ? 'checked' : ''}
              onchange="toggleSelesai(${i})">
            <span style="flex:1">${t.teks}</span>
            <button onclick="hapusTugas(${i})">Hapus</button>
          </div>`
      })

      const sisa = tugas.filter(t => !t.selesai).length
      document.getElementById("info").textContent =
        tugas.length > 0 ? `${sisa} tugas belum selesai dari ${tugas.length} total` : ""
    }

    // Enter juga bisa tambah tugas
    document.getElementById("inputTugas")
      .addEventListener("keypress", (e) => {
        if (e.key === "Enter") tambahTugas()
      })
  </script>
</body>
</html>
Copy semua kode di atas, buka Notepad/VS Code, paste, simpan sebagai todo.html, lalu buka filenya di browser. Aplikasinya langsung jalan!
